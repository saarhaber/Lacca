# Lacca

The Perfect Match for Your Drive

## Live site

**https://saarhaber.github.io/Lacca/**

The site is built and deployed with [GitHub Actions](.github/workflows/deploy-pages.yml). Every push to `main` runs the workflow and publishes the latest `web` build to GitHub Pages.

### One-time repo setup (GitHub)

1. In the repository on GitHub: **Settings → Pages**.
2. Under **Build and deployment**, set **Source** to **GitHub Actions** (not “Deploy from a branch”).
3. Merge or push to `main`; open the **Actions** tab to confirm **Deploy to GitHub Pages** succeeds. The URL above will work after the first successful deploy.
